# вҡҷпёҸ Installation

Clone the repository

```bash
git clone https://github.com/girasepavanraj-dot/bot-league-robotics-frontend.git
```

Go to project

```bash
cd bot-league-robotics-frontend
```

Install dependencies

```bash
npm install
```

Run project

```bash
npm run dev
```

Build project

```bash
npm run build
```
